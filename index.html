Desafio

O desafio proposto pela empresa farmaceutica Sanofi foi relacionado à falta de padronização na nomenclatura de mais de 66 mil substâncias. Essa questão tem um impacto significativo em diversos processos no ecossistema de saúde.

Proposta de solução

Nossa proposta de solução consiste em desenvolver um software capaz de extrair os dados da lista de medicamentos oficial da ANVISA e convertê-los para um dataframe, padronizando a medida de concentração dos fármacos para miligramas (mg).

Além disso, o software irá gerar um novo dataframe com colunas adicionais para que haja a correta classificação das substâncias.

Após a geração do novo dataframe, o software será treinado utilizando a técnica de árvore de decisão (sklearn). Com o treinamento concluído, o usuário poderá selecionar as substâncias desejadas, informar suas concentrações e a inteligência artificial irá apresentar o medicamento igual ou mais próximo que possua os dados informados como entrada.

Informações

  • Regras de negócio

    • Padronização da medida de concentração dos fármacos.

    • O software cria uma coluna de verdadeiro ou falso de acordo com o tipo de medicamento (líquido ou sólido).

    • O output do programa inclui o nome do medicamento, os fármacos presentes e a concentração dos fármacos.

    • O usuário pode buscar apenas um medicamento por vez.

    • O software pode ser acessado de qualquer máquina da empresa.

  • Requisitos funcionais

    • Utilização do PySide6 ou Django + HTML + CSS para a interface.

    • Utilização da árvore de decisão para treinamento da IA.

    • Utilização de um banco de dados para salvar o dataframe.

    • O usuário poderá atualizar o dataframe.

  • Requisitos não funcionais

    • Utilização da linguagem de programação Python.

    • Acurácia do resultado superior a 98%.

    • O software deve realizar a atualização do dataframe com um bom desempenho.
